# Inventory Dashboard  

## 🚀 Technology Used  
- React.js  
- Material-UI  
- Chart.js  
- Axios 
- TailwindCSS
- React Icons  

## 📌 About  
This is a simple admin template for inventory management, providing an intuitive UI to track stock levels, manage products, and generate reports.
